About

I created this site because I had questions about Jeopardy.

The data used here are taken from the excellent Jeopardy site J! Archive. I collected the data and converted it to a JSON file (162M bytes) which the server reads in and searches to get the data. Everything is written in JavaScript and used node.js.

If there are any statistics you want the site to compute send me an email and I will try to add it.
