Overview

This site allows you to perform simple analyses on the results of almost 5000 games of Jeopardy. You can compute from one to four different statistics on the game data at a time. These results are shown in a summary table and as a graph. If you want you call also see a full table of the statistics broken down by individual cell on the 5 rows by 6 categories Jeopary game board.

Filtering the data

The options blocks along the top of the window allow you to specify what you want to count. Each option block is a separate calculation and all four are shown in the summary table, the graph, and the full game board.

Each calculation counts up two numbers, a count and a total. The result is presented as a percent, detemined by dividing the count by the total. Some of the selections in the option block affect only the count and some affect both the count and the total. Let's look at what the selections in an option block mean.

  • # correct: this affects only the count. It will count clues where the number of correct responses matched the value selected. Here is the meaning of each value that can be selected.
    • Do not show: do not show the results of this calulcation in the summary table, the graph, or the full board table.
    • Any: add to the count for every clue, no matter if there was a correct response or not.
    • 0: count the clue if there was no correct response
    • 1: count the clue if there was a correct response
  • # wrong: this affects only the count. It will count clues where the number of wrong responses matched the value selected.
    • Any: add to the count for every clue, no matter how many wrong responses were given.
    • 1-3: count the clue if there was one or more wrong responses
    • 0: count the clue if there was no wrong responses
    • 1: count the clue if there was exactly one wrong response
    • 2: count the clue if there was exactly two wrong responses
    • 3: count the clue if there was exactly three wrong responses
  • Daily doubles: this affects the count and the total. Skip the clue if the daily double specification does not match. Otherwise count and total the clue of the other filters match.
    • Don't care: do not reject any clues.
    • No (exclude): exclude any clues that are daily doubles.
    • Yes (only): only count clues that are daily doubles. Exclude clues that are not daily doubles.
  • Out of order: this affects the count and the total. Skip the clue if the out-of-order specification does not match. Otherwise count and total the clue of the other filters match.
    • Don't care: do not reject any clues.
    • No (exclude): exclude any clues that are out of order.
    • Yes (only): only count clues that are out of order. Exclude clues that are not out of order.
  • Rounds: this affects the count and the total. Skip the clue if the round specification does not match. Otherwise count and total the clue of the other filters match.
    • Don't care: do not reject any clues.
    • Jeopardy: only count clues from the Jeopardy round.
    • Double Jeopardy: only count clues from the Double Jeopardy round.
  • %: this determines the total that is used for the percentage caclulation. The count is always the same but the divisor can change.
    • passed/cell total: the percent is determined by the count and total in the cell.
    • passed/row total: the percent is determined by the count and total in the row.
    • passed/column total: the percent is determined by the count and total in the column.
    • passed/board total: the percent is determined by the count and total in the whole board.
    Note that in the summary table the cell total and the row total are always the same, and the column total and the board total are always the same. In the full board view, with all the rows and columns, the four totals are all different.
  • Counts: for each cell we compute a count and a total. Usually we just display the ratio of these two as a percentage but we can show the count and total also.
    • None: do not show counts or totals.
    • Count only: show the count but not the total.
    • count/total: show the count and the total.
